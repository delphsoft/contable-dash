# Marketing Digital para PyMEs Argentinas

## Filosofía
El marketing digital para PyMEs argentinas tiene restricciones únicas:
inflación que afecta el presupuesto en USD, algoritmos de plataformas globales
que no priorizan el mercado local, y consumidores con alta desconfianza post-2001.
La estrategia debe ser de alto impacto con inversión mínima.

---

## 2. Content Marketing para PyMEs — el modelo que funciona

### Framework: Pillar + Cluster (HubSpot)
```
PILLAR PAGE (contenido amplio):
"Guía completa de compliance fiscal para PyMEs argentinas 2026"

CLUSTER PAGES (contenido específico):
→ "Cómo calcular el IVA de tu negocio"
→ "Monotributo vs RI: cuál te conviene"
→ "Qué es el F931 y cuándo vence"
→ "Cómo validar proveedores en ARCA"
```

Cada cluster page tiene una keyword de long tail con intención de búsqueda alta.
DashBI puede generar este contenido automáticamente con Claude.

### Frecuencia mínima viable para PyME
```
LinkedIn:    3 posts/semana (Juan como thought leader)
Instagram:   4 posts/semana (casos reales + tips fiscales)
WhatsApp:    1 broadcast/semana (vencimiento próximo + tip)
Blog/SEO:    1 artículo/mes (keyword de alto volumen)
```

### Volumen alto, competencia media (atacar primero)
```

"vencimientos afip 2026"                → búsqueda mensual recurrente
"monotributo vs responsable inscripto"  → intención de decisión alta

"f931 vencimiento"                      → búsqueda periódica garantizada

```
"cuánto pago de iva siendo responsable inscripto"
"cómo saber si mi proveedor está en afip"
"cuándo vence el monotributo córdoba"

## 4. MercadoLibre como canal de adquisición

### Por qué es relevante para PymeStudio
- 40M+ usuarios en Argentina
- El vendedor de ML tiene obligaciones fiscales específicas (percepciones ARCA)
- DashBI puede calcular el crédito fiscal de percepciones ML automáticamente

### Estrategia de entrada
```
1. Crear cuenta en ML como vendedor de "servicios digitales"
2. Listar: "Diagnóstico Fiscal PyME - 1 mes DashBI"
3. Precio: ARS equivalente a USD 5 (gancho) → upgrade a plan pago
4. El cliente de ML ya tiene el pain point (percepciones que no sabe recuperar)
```

## 5. Email marketing para PyMEs — secuencia de onboarding

### Secuencia post-diagnóstico (automatizada con Resend)
```
Día 0:  "Tu diagnóstico está listo — score X/100" + PDF adjunto
Día 2:  "La alerta más urgente de tu diagnóstico: [dimensión roja]"
Día 5:  "Cómo [empresa similar] mejoró su score de 45 a 78 en 60 días"
Día 7:  "Próximo vencimiento: [obligación] vence en X días"
Día 14: "¿Implementaste las recomendaciones? Hacé el diagnóstico de seguimiento"
Día 30: "Resumen del mes: [vencimientos cumplidos] + nuevo diagnóstico disponible"
```

### Métricas objetivo
```
Open rate:       > 35% (promedio Latam: 22%)
Click rate:      > 8%
Conversión free→pago: > 15% en 30 días
```
