# Franquicias en Argentina — Estructura Legal, Fiscal y Estratégica

## Contexto
Argentina tiene uno de los mercados de franquicias más desarrollados de Latam,
con más de 1.500 marcas franquiciadas y 35.000+ locales. Es un modelo de
crecimiento especialmente relevante para PyMEs que quieren escalar sin capital.

---

## 1. El modelo de franquicia — fundamentos

### Partes de una franquicia
```
Franquiciante (Franchisor): dueño de la marca y el sistema
Franquiciado (Franchisee):  operador independiente que paga por usar el sistema
```

### Componentes del contrato de franquicia
```
Fee inicial (derecho de entrada):  pago único por acceder a la marca
Regalías (royalties):              % de ventas mensuales (típico 4-8%)
Fondo de publicidad:               % adicional para marketing nacional (1-3%)
Duración:                          típico 3-5 años con opción de renovación
Exclusividad territorial:          zona geográfica asignada
```

---

## 2. Estructura legal para franquicias en Argentina

### No existe ley de franquicias específica en Argentina
El contrato se rige por el Código Civil y Comercial (arts. 1512-1520 contratos
atípicos). Esto da flexibilidad pero también inseguridad jurídica.

### Estructura recomendada para el franquiciante
```
Opción A: SAS franquiciante + contrato de franquicia con cada franquiciado
Opción B: Fideicomiso de marca + SAS operativa
```

**Por qué SAS:**
- Permite múltiples socios con distintas participaciones
- Constitución rápida (online, días)
- Compatible con inversión de terceros

### Registro de marca (INPI) — crítico
```
Antes de franquiciar:
✓ Registrar la marca en INPI (Instituto Nacional de Propiedad Industrial)
✓ Solicitar en todas las clases donde opera el negocio
✓ Tiempo: 18-24 meses para obtener el registro (pero el trámite iniciado da protección)
✓ Costo: ~USD 300-500 por clase
```

**Riesgo sin marca registrada:** el franquiciado podría registrarla y quedarse con ella.

## 3. Estructura fiscal de la franquicia

### Para el franquiciante
```
Fee inicial:   ingreso gravado en Ganancias + IVA al 21%
Regalías:      ingreso mensual gravado en Ganancias + IVA al 21%
Fondo de pub.: puede ser cuenta separada (fideicomiso) sin impacto en Ganancias
```

### Para el franquiciado
```
Fee inicial:   activo intangible amortizable en 10 años (Ganancias)
Regalías:      gasto deducible en Ganancias + crédito fiscal IVA
Local y obra:  activo fijo amortizable
Stock inicial: inventario (no se activa como gasto)
```

### IIBB en franquicias
Si el franquiciante está en Buenos Aires pero el franquiciado en Córdoba:
→ Convenio Multilateral obligatorio para el franquiciante
→ Atribución de ingresos según donde se generan los ingresos

## 4. Manual de operaciones — el activo más importante

El manual de operaciones es lo que diferencia una franquicia de una simple licencia.
Debe incluir:

```
Operaciones: procesos paso a paso, estándares de calidad, proveedores aprobados
Marketing:   lineamientos de marca, materiales, redes sociales
RRHH:        perfil de empleados, capacitación, uniforme
Tecnología:  sistemas requeridos, software, POS
Finanzas:    reportes requeridos, KPIs mínimos, cómo medir la salud del local
Legal:       obligaciones del franquiciado, causales de rescisión
```

**Oportunidad DashBI:** el módulo de compliance DashBI puede ser obligatorio
en el manual de operaciones de la franquicia → PymeStudio cobra por cada local.

## 5. KPIs de una franquicia saludable

### Para el franquiciante (monitor de red)
```
AUV (Average Unit Volume):    ventas promedio por local
Royalty collection rate:       % de regalías cobradas vs devengadas (objetivo: >95%)
Renovación de contratos:       % de franquiciados que renuevan al vencer
NPS franquiciados:             satisfacción de los operadores con el sistema
Tiempo de break-even:          cuántos meses tarda el franquiciado en recuperar inversión
```

### Para el franquiciado (salud del local)
```
Ticket promedio vs benchmark de la red
Costo de mercadería / ventas (CMV) dentro del rango del manual
Costo laboral / ventas dentro del rango
EBITDA del local > umbral mínimo del contrato
Cumplimiento de estándares de marca (auditorías)
```
